# Free Bee #

Free Bee is the free version of the wildly popular Spelling Bee puzzle from the NYTimes.

https://free-bee-puzzle.herokuapp.com/

Try to make as many words as you can from the honeycomb of letters on the board. A word must be at least four letters and must include the central letter. Build longer words to score more points!
